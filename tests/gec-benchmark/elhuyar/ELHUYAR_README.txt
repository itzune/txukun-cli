Train dataset
=============

Each example in the training -and evaluation- datasets we create is
composed of a sentence pair that includes a sentence containing
grammatical errors and its corresponding corrected version. In order
to generate those pairs we apply several rules over grammatically
correct sentences. We also add pairs composed of the original
unmodified sentences so that trained models also take those cases into
account.

Train dataset entries have the following notation:
ORIGINAL_SENTENCE<tab>SENTENCE_WITH_ERRORS

In this resource we include the best performing dataset:

- train/Dt3.tsv: We applied a set of defined rules to each correct
  sentence. From the rules that can be applied, a set of n rules is
  selected at random, and applied sequentially to generate the pairs
  (original -> corrected) and (original -> original). If no rule can
  be applied, only the (original -> original) pair is generated. It
  contains 9,333,672 pairs.


Evaluation dataset
==================

We created two datasets for evaluation purposes. We guarantee a
balance between the pair types None, Single and Multi. In this way, we
generate a first evaluation dataset Dea fully automatically. Taking
into account that, in some cases, the application of the rules can
generate grammatically correct sentences, we also built another Dem
evaluation dataset consisting of a subset of 750 De pairs but reviewed
manually. Pairs generated by the rules that do not really include
grammatical errors are eliminated.

Evaluation dataset entries have the following notation:
ORIGINAL_SENTENCE<tab>SENTENCE_WITH_ERRORS<tab>ERROR_TYPES

We share the following resources:

- evaluation/Dea_none.tsv: Original unmodified sentences with no
  errors. Automatically generated. 2000 pairs.

- evaluation/Dea_single.tsv: Sentences with a single grammatical error
  and their original versions. Automatically generated. 2000 pairs.

- evaluation/Dea_single.tsv: Sentences with multiple grammatical
  errors and their original versions. Automatically generated. 2000
  pairs.

- evaluation/Dem_none.tsv: Original unmodified sentences with no
  errors. Manually revised. 201 pairs.

- evaluation/Dem_single.tsv: Sentences with a single grammatical error
  and their original versions. Manually revised. 250 pairs.

- evaluation/Dem_single.tsv: Sentences with multiple grammatical
  errors and their original versions. Manually revised. 221 pairs.
